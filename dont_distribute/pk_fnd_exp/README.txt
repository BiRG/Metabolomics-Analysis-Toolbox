This directory holds the data etc for my various experiments with peak finding.

This list is already out-of-date, but it will give you an idea of
where to begin.

Files:

flatten_confusion.pl

    Code for making the confusion matrices generated by waffles during my
    machine-learning experiments more machine-parsable.

Directories:

data

    The raw data used in the experiments

data_gen

    The code used to generate the data used in the experiments from
    the synthetic data supplied by Paul Anderson as well as other data
    generating or processing code.  Note that paths in this are
    frequently hard-coded for my experimental system.

<date>

    Data from experiments that took place on (or were started on) a
    particular date
